# ApparelKart

Ecommerce recommendation engine built from scratch

Tech
- Machine Learning Techniques
    - Bag of Words (BoW)
    - TF-IDF
    - Doc2Vec
- Machine Learning  Libraries
    - NLTK, Pandas
    - Numpy, Scikit Learn
- Visualization Tools
    - Matplotlib, Seaborn, Plotly
- Deployment
    - Python, Flask
    - TypeScript, HTML, CSS
    - TailwindCSS
    - Render
